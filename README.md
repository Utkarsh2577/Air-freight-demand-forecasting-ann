
# Air Freight Demand Forecasting using ANN

## 📌 Overview
This project focuses on forecasting air freight demand using time-series data and Artificial Neural Networks (ANN). The model is trained on historical air freight data (2017–2023) to predict future demand patterns.

## ⚙️ Technologies Used
- Python  
- Machine Learning  
- Deep Learning (ANN)  
- Data Analysis & Data Processing  

## 📊 Key Features
- Time-series analysis of freight demand  
- Feature engineering (lag features, rolling averages)  
- Data preprocessing and scaling  
- ANN model for forecasting  

## 📈 Results
- Achieved low prediction error (~3–6%)  
- Successfully captured seasonal trends and demand fluctuations
